# DECISION_LOG

Dieses Logbuch hält wichtige Entscheidungen zum System fest.
Ziel: Nachvollziehbar machen, warum etwas geändert oder freigegeben wurde.

- Jede Entscheidung ist ein eigener Eintrag.
- Einträge werden nicht gelöscht, nur ergänzt.
- Änderungen an Entscheidungen werden als neue Einträge erfasst.
