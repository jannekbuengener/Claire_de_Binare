## brain

Die folgenden Elemente bilden das gemeinsame Systemgehirn:

- CLAUDE.md  
- GEMINI.md  
- AGENTS.md

- GEMINI denkt strukturell.

- Modelle wie COPILOT oder CODEX können diese Umgebung nutzen.  
- Gemeinsam ergibt das das Gehirn.
