# PROMPT_Analysis_Report_Format

Reference: ../AGENTS.md

Usage: Standard format for all agent reports (analysis and delivery).

Template:
```

Context:
Observations / Findings:
Risks (likelihood x impact) and mitigations:
Options / Recommendations (with effort and risk):
Proposed next steps (with owners and order):
Tests/Validation needed:
Open questions:
Decisions required:
```
