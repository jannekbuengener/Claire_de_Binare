# Infrastructure-as-Code definitions and scripts.

## Where to write / Where not to write
*   **Write here:** IaC templates, deployment manifests (e.g., Kubernetes), monitoring configurations, operational scripts.
*   **Do NOT write here:** Application source code, business logic, sensitive credentials (use proper secret management).
