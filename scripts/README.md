# Top-level operational and utility scripts.

## Where to write / Where not to write
*   **Write here:** General-purpose shell or Python scripts for repository-level tasks (e.g., CI-related validations).
*   **Do NOT write here:** Service-specific scripts (use `services/*/`), infrastructure-specific scripts (use `infrastructure/scripts/`), core logic.
